# ChangeLog - Aliyun OSS Media C SDK

## 版本号：1.0.0 日期：2016-03-06
### 变更内容
- 新增H.264,aac转HLS基础接口
- 新增H.264,aac转HLS的录播、直播封装接口
- 优化client，server端接口，提高易用性
- 解决用户无法设置日志级别的问题

## 版本号：0.2.0 日期：2016-01-06
### 变更内容
